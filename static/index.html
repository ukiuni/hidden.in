You might use the SaaS but


You need video chat and screen cast when remote work.

You should better to use internet service.
It's well mentenanced and supported.

But someone (like me) can't use Saas but only use on-premises application,
Use this application "hidden.in".

This site is sample. I recommend using hidden.in on your infrastructure and limited network.
Install is easy.
With docker

$sudo docker run -p 443:3000 ukiuni/hidden.in
or with node 
$git clone https://github.com/ukiuni/hidden.in.git
$cd hidden.in && npm install && npm start

This is open sourced (Apache-2.0) software and easy to start on your server.

You can send issue to GitHub
